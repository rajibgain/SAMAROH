## 🎨 Customize (Optional)

### Change Colors
Edit `src/App.css`:
```css
:root {
  --purple: #your-color;
  --pink: #your-color;
  /* ... */
}
```

### Adjust Animations
Edit `src/styles/*.module.css`:
```css
transition: all 0.3s ease; /* Change 0.3s to your value */
```
