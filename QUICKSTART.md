## 🎉 SAMAROH - Quick Start Guide

Your complete family event coordination platform is ready! Here's how to get started:

### ⚡ Start Using It Now

1. **Run the development server:**
```bash
npm run dev
```

2. **Open in browser:**
- If port 5173 is available: `http://localhost:5173`
- Otherwise: The terminal will show you the correct URL (usually `http://localhost:5174`)

3. **Start creating events!**
   - Click "Create New Event"
   - Enter event name, type, and date
   - Click "Create Event"

### 🎯 First Steps Workflow

1. **Create Event** → Give it a name and date
2. **Add Team Members** → Add family members with roles
3. **Invite Guests** → Start adding guest list
4. **Create Tasks** → Organize what needs to be done
5. **Set Schedule** → Timeline of events
6. **Track Expenses** → Monitor the budget
7. **Use Dashboard** → Monitor progress and statistics

### 📱 Key Features

#### 🎫 Guest Management
- Add guests and their contact info
- Track RSVP status (Pending, Confirmed, Declined)
- View guest statistics

#### ✓ Task Management
- Create categorized tasks (Food, Decoration, Venue, etc.)
- Assign to team members
- Set priorities and due dates
- Mark as complete

#### 💰 Budget Tracking
- Record all expenses
- Categorize spending
- Track who paid
- View budget breakdown

#### 📅 Event Schedule
- Create timeline of events
- Set times and locations
- List attendees per event
- Auto-sorted chronologically

#### 👥 Team Coordination
- Add family members
- Assign specific roles
- Track their responsibilities
- Delegate tasks

#### 📊 Dashboard
- Overview of all metrics
- Guest confirmation status
- Task completion percentage
- Budget summary
- Days until event

### 🎨 How to Navigate

**Main View:**
- **Cards** = Your events (Click to open)
- **+ Create Event** = Add new celebration

**Event View:**
- **Tabs** at top = Navigate between features
- **Dashboard** = Overview of everything
- **Guests** = Manage invitations
- **Tasks** = Track responsibilities
- **Schedule** = Event timeline
- **Budget** = Expense tracking
- **Team** = Family members

### 🐛 Troubleshooting

**Can't see your event?**
- Check internet connection
- Refresh the page
- Clear browser cache

**Changes not appearing?**
- Make sure you're connected to internet
- Check Firebase is working
- Try refreshing the page

**Port already in use?**
- Vite will automatically try the next port
- Check the terminal output for the correct URL
