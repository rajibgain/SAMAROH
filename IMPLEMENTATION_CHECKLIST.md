## 🆘 Troubleshooting

### If AI Bot doesn't appear:
- Check imports in CoordinatorDashboard.jsx
- Verify CSS import
- Clear browser cache
- Check console for errors

### If Vendors tab missing:
- Verify CoordinatorDashboard.jsx changes
- Check import statement
- Reload app

### If Analytics not working:
- Check recharts import
- Verify data is being passed
- Check console for errors

### If Notifications not showing:
- Verify SmartNotificationHub import
- Check that tasks/guests data exists
- Check console errors
